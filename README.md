# Rabi Dahal Portfolio (Plain CSS Edition)

This project is now fully built with:

- HTML
- CSS
- Vanilla JavaScript

No Tailwind CSS dependency is used anywhere in the project.

## Project structure

- `index.html` — semantic page structure
- `styles.css` — complete design system and responsive styling
- `script.js` — interactions (mobile nav, reveal animations, GitHub API data)
- `assets/` — favicon and optional profile image

## Customize quickly

1. Add your LinkedIn URL in the contact section of `index.html`.
2. Replace `your.email@example.com` with your real email.
3. Add your real profile image at `assets/profile-photo.jpg`.
4. Edit sections and project cards directly in `index.html`.
5. Adjust colors and spacing in `styles.css` under `:root`.

## Run locally

Open `index.html` directly in your browser.

## Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "Create personal portfolio"
git branch -M main
git remote add origin https://github.com/proobker/proobker.github.io.git
git push -u origin main
```

Then:

1. Go to **Settings → Pages** in your repo.
2. Set **Source** to “Deploy from a branch”.
3. Choose **main** branch and `/ (root)`.

Site URL:

- `https://proobker.github.io/`
